ПРАВИЛА ИГРЫ "СЛОВА":

1) Вначале игры по середине поля стоит слово на всю ширину поля (поле стационарно - 9*9)
2) Игроки ходят поочереди: добавляя букву к уже имеющимся буквам.  
3) Игрок вводит команду по типу [номер поля]_[буква] (например, A4 п). Так составляются слова
4) Когда на поле появляется новое слово, его длина суммируется уже к имеющейся длина слов
для данного игрока.
5) Наличие слов проверяется только по вертикали (сверху вниз) и по горизонтали (слева направо). 
!!! Если на поле уже есть слово, продолжать его нельзя! (Например, если есть уже слово "ар", то слово "арка" засчитываться уже не будет) 
!!! Все подслова начального слова тоже использовать нельзя (Например, если начально слово "автосалон", то слова "авто"
    "салон" засчитываться не будут
6) Игра заканчивается, когда все поля будут заполнены, либо когда у одного из игроков не будет 10 слов.
7) Выигрывает тот, у кого будет больше суммарная длина всех слов
